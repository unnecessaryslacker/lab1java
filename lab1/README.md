## Мета роботи
Ознайомлення з масивами, матрицями та операціями над ними в мові Java.

## Що реалізовано

- Обчислення C5, C7, C11 за номером залікової книжки
- Генерація матриць різних типів (double, byte, short, int, long, char, float)
- Операції над матрицями залежно від C5:
  - додавання + константа  
  - транспонування  
  - різниця  
  - XOR  
  - множення матриць  
- Обчислення значення над матрицею C залежно від C11
- Усі операції винесено в окремі класи
